## Notes

### Console Output
- `console.log()` is used to print messages to the console or terminal
- Useful for debugging and displaying information

### Variables
- **Variable**: A container for storing data values
- **`let`**: Declares a variable that can be reassigned
- **`const`**: Declares a constant variable that cannot be reassigned
- **`var`**: Older way to declare variables (function-scoped)

### Comments
- Single-line comments: `// comment`
- Multi-line comments: `/* comment */`
